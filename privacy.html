Privacy Policy

Last updated: May 21, 2026

🔒 Classroom Plus does not collect, store, transmit, or sell any personal data to external servers. All data stays in your browser.

1. Overview

Classroom Plus ("the Extension") is a Chrome browser extension that enhances Google Classroom with search, bookmark, and navigation features. This privacy policy explains what data the Extension accesses, how it is used, and how it is stored.

2. Data We Access

The Extension requests the following Google API permissions to provide its features:

  • Classroom Courses (read-only): Used to retrieve a list of your active courses so the Extension can identify which class you are viewing.
  • Classroom Announcements (read-only): Used to fetch stream posts and announcements for the search feature.
  • Classroom Coursework (read-only): Used to fetch assignments and coursework for the search feature.
  • Classroom Course Work Materials (read-only): Used to fetch class materials for the search feature.
  • Google Drive (read-only): Used exclusively for the "Deep PDF Search" feature. When you initiate a deep search, the Extension reads the content of PDF and document files attached to your Classroom posts to search for text within them.

3. How Data Is Used

All data retrieved from Google Classroom and Google Drive APIs is used solely for the following purposes:

  • Displaying search results for posts, assignments, and materials within your Google Classroom classes.
  • Extracting text from attached documents to enable deep search within file content.
  • Storing your bookmarks so you can quickly access saved posts.

4. Data Storage

The Extension stores data exclusively in your browser's local storage using the Chrome Storage API. This includes:

  • Your bookmarked posts.
  • Cached search results and extracted document text (to speed up repeated searches).
  • Your dark mode preference.
No data is ever transmitted to, stored on, or processed by any external server, third-party service, or analytics platform.

5. Data Sharing

Classroom Plus does not share, sell, rent, or transfer any user data to any third parties. Period.

6. Data Retention

Cached data (search results and extracted document text) is stored locally and can be cleared at any time by:

  • Uninstalling the Extension (which removes all local data).
  • Clearing your browser's extension storage via Chrome settings.

Bookmarks are retained until you manually delete them or uninstall the Extension.

7. Security

The Extension uses Google's OAuth 2.0 authentication system to securely access your Classroom and Drive data. The Extension never sees or stores your Google password. Authentication tokens are managed entirely by Chrome's built-in identity system.

8. Children's Privacy

Classroom Plus is designed for use by students and educators. The Extension does not knowingly collect any personal information beyond what is described in this policy. Since all data is stored locally in the user's browser and never transmitted externally, no personally identifiable information is collected or processed by us.

9. Google API Services User Data Policy

Classroom Plus's use and transfer to any other app of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. Continued use of the Extension after changes constitutes acceptance of the updated policy.
